# 志远 · AI时代高考择业助手

用六维智能评估框架帮高考学生在AI时代做出有预见性的专业选择。

## v3.0 六维评估框架

1. **AI冲击评估**：AI生存力 + 人机协作增值 + 转型弹性
2. **窗口期评估**：政策信号 + 国际趋势 + 就业 + 薪资 + 人才缺口
3. **家庭适配**：经济支持 + 行业资源 + 风险偏好 + 城市 + 时间 + 出国 + 优先级 + 排除
4. **人格匹配**：霍兰德RIASEC + MBTI E/I T/F + 大五人格（OCEAN）
5. **国家战略**：十五五规划契合度 + 专业存续安全 + 人才缺口规模
6. **可迁移性**：跨行业通用性 + 考研灵活度 + 交叉发展潜力

## 核心技术

- **TOPSIS 多准则决策排序**：替代简单加权求和，同时考虑"离最佳有多近"和"离最差有多远"
- **Fuzzy AHP 个性化权重**：根据用户的 SCCT 结果期望自动推导六维权重
- **大五人格推导**：从 15 题行为问卷推导开放性、尽责性、外向性、宜人性、情绪稳定性
- **SCCT 社会认知**：自我效能 + 结果期望 + 目标清晰度 → 影响推荐方向
- **DeepSeek AI**：个性化报告（7 部分含 Plan B）、10 年后场景模拟、一句话摘要

## 功能列表

- 正向测评：走问卷 → 六维 TOPSIS 排序 → 个性化推荐
- 反向测评：输入心仪专业 → 六维适合度分析
- 完整报告：1500字七部分职业规划报告（含 Plan B 备选）
- 10年后的你：第一人称未来场景叙述
- 风险预警：低分维度自动提示 + 对冲建议
- 避雷榜：专业避雷榜（warning.html）
- 家庭分歧诊断：父母和孩子各填一遍，AI 调解建议（conflict.html）
- 家长视角 / 学生视角 切换
