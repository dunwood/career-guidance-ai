# 择业罗盘 — AI高考志愿指导助手

用三维框架帮高考学生在AI时代做出有预见性的专业选择。

## 三维框架
1. **AI生存分**：这份工作的本质是「识别模式」还是「处理例外」？
2. **窗口期分**：4年后毕业，是赶上浪峰还是浪尾？
3. **人格匹配分**：你的行为特质和这个方向的日常工作是否契合？

## 文件结构

```
career-guidance-ai/
├── prompts/
│   └── system.md          ← 系统提示词，粘贴到Coze的System Prompt框
├── data/
│   ├── majors.csv         ← 专业数据，上传到Coze知识库
│   └── majors-template.csv ← 新增专业时用这个模板
└── README.md
```

## 数据更新流程

### 平时（每月5分钟）
扫一眼重大政策新闻，有变化时直接在GitHub网页上编辑 `data/majors.csv`，
更新对应专业的「窗口期说明」和「last_updated」字段。

### 集中更新（每年1-3月，约2天）
各大报告发布后集中更新：
- 麦可思就业报告（2月）
- 智联/猎聘薪酬报告（1-2月）
- 教育部就业数据（3月）

### 更新后同步到Coze
1. 在GitHub编辑并提交 `data/majors.csv`
2. 打开Coze → 知识库 → 删除旧文件 → 上传新的 `majors.csv`
3. 完成

## 新增专业
复制 `data/majors-template.csv` 里的空行，填写所有字段，加到 `majors.csv` 末尾。
